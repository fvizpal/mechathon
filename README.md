## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/fvizpal/mechathon.git
```

2. Intall dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
